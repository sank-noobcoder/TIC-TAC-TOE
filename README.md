# 🌌 Neon Night Tic-Tac-Toe

A visually striking, responsive Tic-Tac-Toe game featuring a deep-space aesthetic, animated shooting stars, and an integrated computer opponent.

 ## ✨ Features

* **Play vs Computer:** An automated opponent that picks moves randomly from available slots.
* **Night Sky Aesthetic:** A dynamic background with a twinkling star field and CSS-animated shooting stars.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using CSS Grid and `clamp()` typography.
* **Neon UI:** High-contrast cyan interface with glowing effects and a "winner strike-through" logic.
* **Clean Reset:** A dedicated reset button to clear the board and restart the game state without refreshing.

## 🚀 Tech Stack

* **HTML5:** Semantic structure for the game board.
* **CSS3:** Advanced animations (keyframes), Flexbox, Grid, and Custom Properties for the space effect.
* **JavaScript (ES6):** Game logic, win-pattern detection, and computer move automation.
